# raizes-e-ritmos

O projeto é para as pessoas participarem dos eventos, conhecerem como é o nosso objetivo, serviço, público-alvo, tirarem dúvidas e conhecerem um pouco as músicas culturais Brasileiras.

## Funcionalidades
No código temos acessibilidade para surdos e para quem tem baixa visão. Temos fotos para verem um pouco de alguns eventos e como as pessoas tirarem dúvidas.

## Tecnologias e Acessibilidade
*   **Vlibras:** Integração para acessibilidade em libras.
*   **Contraste e Legibilidade:** Ajustes com a cor amarela para facilitar a leitura para quem tem baixa visão.
